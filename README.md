# 📝 ToDo List Application

## 🚀 프로젝트 개요
React.js v18를 사용하여 개발된 **ToDo 리스트 애플리케이션**입니다.  
사용자는 할 일을 **추가, 삭제, 완료 처리**할 수 있으며,  
**'진행 중'과 '완료' 목록을 탭을 통해 관리**할 수 있습니다.

## 📌 주요 기능
✅ **할 일 추가 / 삭제 / 완료 처리**  
✅ **진행 중 & 완료 리스트 탭 분리**  
✅ **할 일 수정 기능**  
✅ **로컬 스토리지 연동** (새로고침해도 데이터 유지)
